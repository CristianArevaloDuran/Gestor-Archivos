# 📁 Gestor de Archivos

Aplicación web que permite a los usuarios explorar, subir, descargar y gestionar archivos.

## 🛠️ Tecnologías Utilizadas

### Frontend
- **Next.js**
- **Tailwind CSS**
- **Axios** 

### Backend

- **Node.js**
- **Express**
- **MongoDB**
